Tasy Metadata Extension

Chrome Extension to present Tasy metadata information.

Features

  • Displays the attribute and value of the fields.
  • Displays the attribute of each column in the grid.
  • Displays the panel details like code, view and table.
  • Allows to copy attribute name, schematics code, view code, etc, when clicked.
  • Displays recent features opened for quick access.
  • The mouse over a recent feature displays on the tooltip the code and name of that feature.
  • The extension updates the name of the recent features according to the user's current language. So, you can change it, log in again and see the translation.
  • Reload stylesheets without reload the whole page.
  • On popup, displays the WHEBDC network IP.

Inspect mode

The inspect mode is a powerful tool to discovery some aspects of any component in the page, inspecting anything e taking information about:

Feature The code and the name of the current feature.
Element See the inspected element HTML, allowing to see the source image of a <img>, the code attribute of buttons and more.
Panel Properties like code, view, table, object type, action name and more.
Popup menus See the code of each menu component in the inspected panel.
Popup items See the items three in the inspected popup menu.
Popup single item Details of inspected item in the popup menu.
Filter configurator Details of the filter.
Attributes List of all attributes in the panel (for dbpanel).
Meta attributes and
Meta columns		 Detailed list of attributes and columns in the panel.
Attribute The name, value, type, table, options (for radio buttons) and more, when
inspecting an attribute like a locator, listbox or even a simple text input.
Attribute options List with the code and description of each option available for selecting
in that attribute (for listbox).
Tabs The code and the name of each tab in the panel.
Calendar Displays its action, date, type and parentDtoObject of the inspected
calendar component.

Install

  1. Choose an option below to get the extension files:

    • Download ZIP, or
    • \\whebs05\Tecnico\Programas\Instalador oficial HTML5\tasy-metadata-extension\tasy-metadata-extension.zip

  2. Open this page on Chrome: chrome://extensions

  3. Select Developer mode.

  4. Click in the Load unpacked button.

  5. Select dist folder inside extension's folder.

  6. Pin the extension on toolbar, it's easier to access.

Obs: to enable the extension in incognito mode, open chrome://extensions, go to Details on extension and check Allow in incognito.

Updating

To update the extension you must download the latest version following the Install instructions above, then extract the files to the same folder overriding old files, then refresh the extension on Chrome's extensions page.

Usage

In a page from Tasy domain or localhost:3000, log in, open any feature and click in the extension icon on toolbar to select an option and see the effects.

You can configure in what hosts the extension should be available, to do that you can right click in the extension icon on toolbar, go to "Options" and add a newline with the host that you want. After change the list click in the "Save" button and reload the page.
